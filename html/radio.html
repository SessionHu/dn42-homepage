Playlist (unordered)

If you have some good musics but they are not in the playlist. Please feel free to contact us, sending us your music files and short description. We will add to the playlist if I also like them.

We prefer 48kHz 192kb/s Ogg Vorbis with metadata (our current output configuration), but FLAC, M4A, MP3 and other open/free format are also accepted. You have better use FFmpeg (or other free/libre media tools, e.g. Gstreamer) to convert your file to our recommended configuration if it is too big (greater than 50MiB), since our server storage is not infinity. Please note that we do not accept any non-free encrypted music file format.
