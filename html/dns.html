Recursive Name Servers

Servers

This server is accessible via both clearnet and dn42 from all IPs, and will return results for any dn42 and clearnet domain names that can be resolved.

You need to contact us to get the clearnet IP address and/or domain name for security concerns. Please note that our resources are not unlimited, and we reserve the right to ban your IP address if the service is abused.

Additionally, please note that TLS and HTTPS certificates are issued by Let's Encrypt for the subject xhustudio.eu.org and subjectAltName *.xhustudio.eu.org. So do not be surprised if you encounter a certificate warning when using TLS or HTTPS via dn42.
