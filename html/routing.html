SESS.DN42 Routing Policy

SessNetwork (AS4242422466) uses a custom routing method designed for dn42 internet. We call it Poisoned Potato Routing. Our bgp_local_pref base value is 200 for eBGP, 160 for iBGP.

BGP Communities

We support some well-known and common communities. They are defined in RFCs and dn42 wiki.

NO_EXPORT(65535, 65281)Reject in eBGP export filter
NO_ADVERTISE(65535, 65282)Reject in iBGP and eBGP export filter
NO_EXPORT_SUBCONFED(65535, 65283)Reject in eBGP export filter
BLACKHOLE(65535, 666)Set destination to RTD_BLACKHOLE in iBGP and eBGP import filter
(64511, DN42_RTT)(64511, 1..9)Add in iBGP and eBGP, import and export filter. Will be modified according to your RTT. bgp_local_pref += 9 - DN42_RTT
(64511, DN42_LINK)(64511, 31..34)Add in iBGP and eBGP, import and export filter. Will be modified according to your link security. bgp_local_pref += DN42_LINK - 31
(64511, DN42_REGION)(64511, 41..70)Add in iBGP and eBGP, export filter, for prefixes originated from AS4242422466. bgp_local_pref += 40 in eBGP import filter
(64511, DN42_COUNTRY)(64511, 1000..1999)Add in iBGP and eBGP, export filter, for prefixes originated from AS4242422466. bgp_local_pref += 20 in eBGP import filter
